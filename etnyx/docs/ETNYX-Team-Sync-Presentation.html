👤 Step-by-Step: Customer Order

Apa yang customer lihat & lakukan dari awal sampai akhir

Step 1 — Pilih Paket
Buka etnyx.com/order
Pilih mode: Paket (rank→rank), Per-Star (star aja), atau Gendong (mabar)
Lihat harga dari pricing catalog (CMS)
Opsi: Express (+20%), Premium (+30%)
Masukkan promo code (kalau ada)
Step 2 — Data Akun
Isi User ID + Server ID (Moonton format)
Sistem verifikasi akun otomatis via Moonton API
Pilih login method (Moonton/FB/Google/TikTok/VK/Apple)
Masukkan login & password → dienkripsi AES-256 sebelum simpan
Request hero favorit (opsional)
Step 3 — Kontak & Add-ons
Isi nomor WhatsApp & email
Pilih opsi Express / Premium
Tulis catatan khusus untuk worker
Apply promo code (kalau ada)
Step 4 — Bayar
Pilih: Auto Payment (Midtrans — QRIS, VA, e-wallet) atau Manual Transfer (11 metode bank/e-wallet)
Auto → redirect ke Midtrans, bayar, otomatis confirm
Manual → lihat rekening tujuan, transfer, upload bukti
Setelah order: Customer otomatis dapat WA + Email konfirmasi dengan link tracking — bisa pantau progress kapan saja tanpa harus tanya CS.

⚡ Apa yang Otomatis vs Manual

✅ Otomatis (Sistem Handle)

Order creation + validation
Akun verification (Moonton API)
Credential encryption (AES-256)
Order ID generation (ETX-YYMMDD-###)
Payment auto-confirm (Midtrans webhook)
WA notif ke customer (setiap status change)
Telegram notif ke admin/worker group
Email konfirmasi
Tracking page real-time (public)
Progress auto-update dari worker submission
Invoice auto-generate
Auto-testimonial dari review ≥4⭐
Reward points auto-credited
Promo code validation + discount calc
Tier discount auto-apply
Rate limiting + security blocking
Audit trail (semua action tercatat)

⚠️ Manual (Butuh Manusia)

Verify bukti transfer manual → Admin
Assign worker ke order → Leader
Push rank (main game) → Worker
Upload progress + screenshots → Worker
Mark order selesai → Worker
Handle komplain / eskalasi → CS / Admin
Approve testimonial featured → Admin
Send manual follow-up (optional) → CS
Quality check worker → Leader
Update pricing / promo → Admin
Takeaway: 17 proses otomatis vs 10 manual.
Manual pun rata-rata cuma klik 1-2 tombol di dashboard.

⭐ Scope PIC: Leader (2 orang)

Dashboard: /admin/lead

📋 Assign Worker ke Order
Tugas paling krusial Leader.
• Lihat order status "confirmed" (sudah bayar, belum di-assign)
• Pilih worker yang tepat (sesuai kapasitas, rank, availability)
• Assign via Lead Dashboard → worker otomatis dapat notif Telegram
• Tulis notes untuk worker (instruksi khusus, hero request, jam main)
✅ Quality Assurance
• Cek progress submissions worker (screenshots, win rate)
• Pastikan SOP keamanan dipatuhi (no emulator, VPN sesuai, dll)
• Spot-check: credentials aman, tidak ada aktivitas mencurigakan
• Review worker performance secara berkala
🤝 Koordinasi & Eskalasi
• Rotasi worker kalau overload / unavailable
• Handle replacement kalau worker ga perform
• Eskalasi masalah ke Admin (komplain, refund, dll)
• Mediasi worker-customer issue
📊 Reporting
• Laporan harian: order aktif, worker aktif, order pending assign
• Laporan mingguan: completed, issues, worker stats
• Training worker baru soal SOP & dashboard

💬 Scope PIC: CS (1 orang)

Tool utama: WhatsApp Business + Dashboard (view only)

📲 Handle Chat Masuk
• Jawab DM dari sosmed / WA → SLA: < 5 menit
• Kualifikasi: tanya nama, rank sekarang, target rank
• Kasih info harga (refer ke pricing catalog)
• Guide customer ke etnyx.com/order untuk order sendiri
• Prinsip: arahkan ke self-service, bukan manual orderin
🛡️ Handle Objection
• "Mahal" → Bandingin push sendiri vs 2-3 hari di ETNYX
• "Takut ban" → Jelaskan enkripsi, VPN, play natural, 0 banned
• "Pikir-pikir" → Hold slot + harga 2 jam, season reset
• "Garansi?" → Star turun = gratis push ulang, refund sebagian
🔄 Follow-up
• +2 jam: "Masih tertarik push rank?"
• +24 jam: "Promo masih berlaku hari ini"
• +3 hari: "Kalau butuh bantuan, ETNYX siap!"
• Max 3x follow-up, jangan spam
📞 Handle Komplain
• Customer complaint → cek status di tracking page
• Kalau butuh action → eskalasi ke Leader / Admin
• Kalau update progress → info "bisa cek di link tracking"
• Log setiap interaksi penting
Key point: Karena customer bisa track sendiri, beban CS berkurang drastis. CS fokus di closing leads baru dan handle exception, bukan kasih update order.

📸 Update Progress
1. Setiap milestone → isi form: stars gained, matches, win/mvp/savage

3. Submit → customer langsung lihat update di tracking page
4. Bisa edit submission dalam 30 menit
5. Target selesai → klik "Selesai" → customer auto dapat WA
