5 Content Pillars — Posting Distribution

😤 PROBLEM (20%)

Goal: Relate, trigger pain
Format: Short video, meme
Topics: Solo queue toxic, stuck rank, skill gap
Posting: Monday-Friday 12:00

😂 ENTERTAINMENT (20%)

Goal: Viral, reach, shares
Format: Funny clip, meme, dance
Topics: ML meme, funny moments, trending
Posting: Tuesday-Thursday 18:00

📚 EDUCATION (20%)

Goal: Trust, authority
Format: Tutorial, tips & tricks
Topics: Hero guide, rank push tips, build item
Posting: Carousel 2x/minggu

🏆 PROOF (25%)

Goal: Credibility, FOMO
Format: Testimoni, before-after, rank proof
Topics: Client results, mythic push, success
Posting: Story 3x/hari

💰 SELLING (15%)

Goal: Conversion, CTA
Format: Promo, how-to-order, limited offer
Topics: Flash sale, "order sekarang", bonus
Posting: Story peak hours

✅ Pre-Posting Checklist

Hashtag: 15-20 hashtag (mix: reach + niche + trending)
Link Bio: WhatsApp link active? Bisa diklik?
Visual: Subtitle jelas? Sound on-brand? 1080x1920 dimension OK?
CTA Visible: "DM", "Chat WA", "Follow", "Save" ada?
Scheduled Time: Posting at optimal time (12:00 / 18:00)?
File Naming: Format: DayXX-Theme-Title.mp4
Backup: Original file saved? Ke cloud? (Google Drive / Dropbox)
