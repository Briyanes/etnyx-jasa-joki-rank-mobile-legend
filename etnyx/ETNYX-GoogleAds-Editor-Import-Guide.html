📋 Quick Start Guide

Files Available:

  • ETNYX-GoogleAds-Editor-Compliant.csv - Main import file (keywords + headlines + descriptions)
  • ETNYX-GoogleAds-Compliant-Sitelinks.csv - Separate sitelinks file (campaign-level)

STEP 1: Download & Open Google Ads Editor

  1. Download Google Ads Editor on your computer
  2. Open Google Ads account
  3. Go to File → Import → Campaign from file

STEP 2: Import Main Campaign File

  1. Select: ETNYX-GoogleAds-Editor-Compliant.csv
  2. Choose: "Create new campaign"
  3. Campaign name: [Search] ETNYX — High Intent ML Coaching
  4. Click "Next" → "Import"

STEP 3: Add Campaign-Level Sitelinks

  1. Right-click campaign name
  2. Select "Edit"
  3. Go to "Sitelinks" section
  4. Click "+ Sitelink"
  5. Add sitelinks from table below

STEP 4: Review & Post

  1. Review all keywords, headlines, descriptions
  2. Verify Final URLs (should all be: https://etnyx.com/order)
  3. Click "Post" to push to live
  4. Monitor approval (should take 24 hours)
