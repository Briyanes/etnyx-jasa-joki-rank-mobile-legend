🚀 Proses Import Step-by-Step

1Persiapan

  • Download Google Ads Editor ke komputer (jika belum terinstall)
  • Buka akun Google Ads dan login
  • Siapkan file CSV untuk download

2Import Kampanye Utama

  • Di Google Ads Editor, klik File → Import → Campaign from file
  • Pilih: ETNYX-GoogleAds-Editor-Compliant-ID.csv
  • Klik Open
  • Pilih "Create new campaign"
  • Nama kampanye auto-fill: [Search] ETNYX — Coaching ML Niat Tinggi
  • Klik Import → Tunggu selesai

3Tambah Sitelink (Manual atau Import)

Opsi A: Entry Manual (Rekomendasi untuk setup pertama kali)

  • Klik kanan nama kampanye: [Search] ETNYX — Coaching ML Niat Tinggi
  • Pilih Edit campaign
  • Scroll ke bagian "Sitelinks"
  • Klik + Sitelink
  • Tambah setiap sitelink dari tabel (17 total):
  • Link Text: Coaching per Rank
  • URL: https://etnyx.com/order
  • Description 1: Coaching personal untuk rank Anda
  • Description 2: Pilih paket training
  • Ulangi untuk semua 17 sitelink

Opsi B: File Import

  • Klik Tools → Import → Extensions
  • Pilih ETNYX-GoogleAds-Compliant-Sitelinks-ID.csv
  • Ikuti wizard import

4Review Semua Element

  • Expand kampanye dan klik setiap grup iklan
  • Verifikasi 4 grup iklan:
    • ml-core (5 keyword)
    • pelatihan-rank (5 keyword)
    • tier-spesifik (5 keyword)
    • duo-coaching (5 keyword)
  • Cek semua keyword ada dan benar
  • Verifikasi headline (harus ada 15 headline unik)
  • Cek deskripsi (4 deskripsi)
  • Konfirmasi Final URL: https://etnyx.com/order

5Post ke Live

  • Klik tombol Post (kanan bawah)
  • Konfirmasi post
  • Tunggu selesai (biasanya ~1-2 menit)
  • Cek akun Google Ads untuk memastikan kampanye muncul

6Monitor Persetujuan

  • Buka akun Google Ads → tab Ads
  • Status harus menunjukkan "Eligible" atau "Pending review"
  • Setelah 24 jam, harus menunjukkan "Enabled" + "Approved"
  • Jika ditolak, cek alasan penolakan dan eskalasi

❓ Pertanyaan Umum

T: Bisa import sitelink bersamaan dengan kampanye?

J: CSV import utama (ETNYX-GoogleAds-Editor-Compliant-ID.csv) sudah include sitelink, tapi mungkin perlu review manual. Untuk setup lebih rapi, tambah sitelink setelah kampanye import selesai.

T: Kalau hanya mau import 1-2 grup iklan saja?

J: Edit file CSV dan hapus baris untuk grup yang tidak mau. Jaga header row + baris grup yang diinginkan.

T: Bisa ubah headline setelah import?

J: Ya! Google Ads Editor allow edit headline kapan saja sebelum post. Setelah post, bisa edit di web interface Google Ads.

T: Berapa lama sampai iklan muncul setelah post?

J: Biasanya 24-48 jam untuk Google review dan approve. Setelah approve, iklan bisa muncul dalam hitungan jam (tergantung budget/bid).

T: Kalau iklan ditolak lagi?

J: Cek alasan penolakan. Semua aset ini sudah verified sesuai kebijakan, tapi kalau Google flag sesuatu, masalahnya mungkin dari konten landing page atau riwayat akun. Hubungi Google support untuk appeal.

🔧 Troubleshooting

Masalah: CSV tidak bisa import

  • Cek format file adalah .csv (bukan .xlsx atau .xls)
  • Pastikan file punya header yang benar (Campaign, Ad Group, Keyword, dll)
  • Coba buka CSV di Excel dulu untuk verifikasi format

Masalah: Headline tampil "incomplete"

  • Normal - responsive search ads bisa punya 15 headline tapi minimum biasanya 3-5
  • Google akan rotate semua 15 otomatis

Masalah: Sitelink tidak tampil di iklan

  • Pastikan sitelink disave di level KAMPANYE, bukan grup iklan
  • Sitelink butuh 1-2 hari untuk tampil di iklan
  • Cek status kampanye "Enabled" bukan "Paused"

Masalah: Final URL tidak bisa terima path custom

  • Semua URL di CSV ini pakai https://etnyx.com/order
  • Untuk ganti ke URL beda, edit kolom "Final URL" di CSV sebelum import
  • Setelah import, bisa juga modify di Google Ads Editor
