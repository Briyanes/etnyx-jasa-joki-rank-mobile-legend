🚀 Step-by-Step Import Process

1Prepare

  • Download Google Ads Editor to your computer (if not already installed)
  • Open Google Ads account and sign in
  • Have CSV files ready to download

2Import Main Campaign

  • In Google Ads Editor, click File → Import → Campaign from file
  • Select: ETNYX-GoogleAds-Editor-Compliant.csv
  • Click Open
  • Choose "Create new campaign"
  • Campaign name auto-fills: [Search] ETNYX — High Intent ML Coaching
  • Click Import → Wait for completion

3Add Sitelinks (Manual or Import)

Option A: Manual Entry (Recommended for first-time setup)

  • Right-click campaign name: [Search] ETNYX — High Intent ML Coaching
  • Select Edit campaign
  • Scroll to "Sitelinks" section
  • Click + Sitelink
  • Add each sitelink from table (14 total):
  • Link Text: Coaching by Rank
  • URL: https://etnyx.com/order
  • Description 1: Personalized coaching for your rank
  • Description 2: Choose training plan
  • Repeat for all 14 sitelinks

Option B: File Import

  • Click Tools → Import → Extensions
  • Select ETNYX-GoogleAds-Compliant-Sitelinks.csv
  • Follow import wizard

4Review Everything

  • Expand campaign and click each ad group
  • Verify 4 ad groups:
    • ml-core (5 keywords)
    • rank-training (5 keywords)
    • tier-specific (5 keywords)
    • duo-coaching (5 keywords)
  • Check all keywords are present and correct
  • Verify headlines (should see 15 unique headlines)
  • Check descriptions (4 descriptions)
  • Confirm Final URL: https://etnyx.com/order

5Post to Live

  • Click Post button (bottom right)
  • Confirm posting
  • Wait for push to complete (usually ~1-2 minutes)
  • Check Google Ads account for campaign appearance

6Monitor Approval

  • Go to Google Ads account web → Ads tab
  • Status should show "Eligible" or "Pending review"
  • After 24 hours, should show "Enabled" + "Approved"
  • If disapproved, check disapproval reason and escalate

❓ Common Questions

Q: Can I import sitelinks at the same time as campaign?

A: The main CSV import (ETNYX-GoogleAds-Editor-Compliant.csv) includes sitelinks, but they may need manual review. For cleaner setup, add sitelinks after campaign import.

Q: What if I only want to import 1-2 ad groups instead of all 4?

A: Edit the CSV file and delete rows for ad groups you don't want. Keep header row + rows for your desired groups.

Q: Can I change the headlines after import?

A: Yes! Google Ads Editor allows editing headlines anytime before posting. After posting, you can edit live campaigns through Google Ads web interface.

Q: How long until ads show up after posting?

A: Usually 24-48 hours for Google to review and approve. After approval, ads can start showing within hours (depending on budget/bids).

Q: What if ads get disapproved again?

A: Check the disapproval reason. All these assets are verified compliant, but if Google flags something, the issue is likely landing page content or account history. Contact Google support for appeal.

🔧 Troubleshooting

Issue: CSV won't import

  • Check file format is .csv (not .xlsx or .xls)
  • Ensure file has proper headers (Campaign, Ad Group, Keyword, etc.)
  • Try opening CSV in Excel first to verify formatting

Issue: Headlines show as "incomplete"

  • This is normal - responsive search ads can have 15 headlines but minimum is usually 3-5
  • Google will rotate through all 15 automatically

Issue: Sitelinks not appearing on ads

  • Make sure sitelinks are saved at CAMPAIGN level, not ad group level
  • Sitelinks take 1-2 days to appear on ads
  • Check campaign status is "Enabled" not "Paused"

Issue: Final URL won't accept custom path

  • All URLs in this CSV use https://etnyx.com/order
  • To change to different URL, edit CSV column "Final URL" before import
  • After import, you can also modify in Google Ads Editor
