🔐 Meta CAPI (Server-Side)

Selain client-side pixel (browser), ETNYX juga mengirim Purchase event via Meta Conversions API (server-side):
  • Bypass ad blockers — server-side tidak terblok adblocker
  • Event deduplication — via event_id, Meta tau ini event sama
  • PII hashing — email & phone di-hash SHA-256 sebelum dikirim
  • File: lib/meta-capi.ts → trigger setelah payment webhook sukses

6. Budget & Scaling Roadmap Advertiser

TL;DR: 4 fase. Mulai Rp 50K/hari → test 2 minggu → optimize → scale ke Rp 300K+/hari di bulan 3. Naikkan budget 20% per 2-3 hari — JANGAN loncat >50%.

📅 Phase 1: Testing (Minggu 1-2) — Rp 50K/hari

Funnel%DailySetup
TOF70%Rp 35K1 Campaign, 3-5 Ad Sets (beda audience), 2-3 creative per ad set
MOF20%Rp 10K1 Campaign, 1-2 Ad Sets (IG engagers + web visitors)
BOF10%Rp 5K1 Campaign, 1 Ad Set (cart + checkout abandoners)

Goals Phase 1

  • Test 6-10 creative variations (hook, angle, format berbeda)
  • Kumpulkan min 50 pixel events (ViewContent, AddToCart)
  • Identifikasi winning creative (CTR > 2%, CPC < Rp 3.000)
  • Build custom audiences dari pixel data
  • Target CPA: Rp 30.000 – 50.000 per purchase
⏳ Sabar! Di budget Rp 50K/hari, Meta butuh 3-5 hari learning phase. Jangan matikan ad set sebelum 3 hari kecuali spend >2x target CPA tanpa konversi.

📅 Phase 2: Optimization (Minggu 3-4) — Rp 75-100K/hari

Funnel%DailyAction
TOF60%Rp 45-60KKill losers, duplicate winners, test new creatives
MOF25%Rp 19-25KAdd video viewers, test carousel social proof
BOF15%Rp 11-15KDynamic retargeting, urgency creatives

Optimization Rules

  • Kill: Ad set CPA > 2x target setelah 3 hari
  • Scale: Ad set CPA ≤ target → naikkan budget 20% per 2-3 hari
  • Duplicate: Winning ad set → copy ke audience berbeda
  • Refresh: Creative fatigue setelah 7-10 hari (frequency > 2.5) → ganti hook/visual
  • Mulai buat LAL dari purchase data (setelah ≥ 50 purchases)

📅 Phase 3: Scaling (Minggu 5-8) — Rp 150-300K/hari

Funnel%DailyAction
TOF55%Rp 83-165KLAL Purchasers 1%+3%, expand geo, Advantage+
MOF25%Rp 38-75KFull retarget stack, sequential storytelling
BOF20%Rp 30-60KPromo codes, limited offers, repeat buyer

Scaling Strategy

  • Horizontal: Duplicate winning campaign ke audience segments baru
  • Vertical: Naikkan budget 20-30% setiap 3 hari (jangan >50%)
  • Advantage+: Setelah 100+ purchases, test ASC (algo full optimize)
  • CBO: Biarkan Meta alokasikan budget ke ad set terbaik
  • Target ROAS: Min 3x (spend 100K → revenue 300K)

📅 Phase 4: Maturity (Bulan 3+) — Rp 300K-1jt/hari

  • Full Advantage+ Shopping Campaign sebagai main driver
  • Retarget purchasers untuk repeat order (upsell paket lebih tinggi)
  • Seasonal push: naikkan budget 2-3x saat season reset ML
  • Cross-platform: parallel TikTok Ads & Google Ads
  • Test influencer/KOL collab (micro-influencer ML 5-50K followers)
  • Target CPA: Rp 15.000-25.000
  • Target ROAS: 4-6x

11. IG & FB Organic Synergy Editor

Untuk Editor: Konten organic di IG adalah "testing ground" gratis. Post yang perform bagus → jadikan ad creative → sudah terbukti works.

📱 Instagram @etnyx.id Strategy

  • Post organic 3x/minggu (sesuai 9-Post Grid plan) → jadi creative bank untuk ads
  • Boost best-performing posts → jadikan ad via "Boost Post" atau create ad dari existing post
  • Stories daily: progress update, BTS push rank, promo countdown
  • Reels first: 80% konten = Reels (reach terbesar di IG 2026)
  • CTA di setiap post: "Link di bio" atau "DM untuk order"
  • Engagement bait: Poll di Stories ("Main role apa?"), Quiz, Q&A
