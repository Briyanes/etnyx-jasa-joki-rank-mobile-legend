🔐 Meta CAPI (Server-Side)

Selain client-side pixel (browser), ETNYX juga mengirim Purchase event via Meta Conversions API (server-side). Ini penting karena:

  • Bypass ad blockers — server-side tidak terblok adblocker
  • Event deduplication — via event_id, Meta tau ini event sama
  • PII hashing — email & phone di-hash SHA-256 sebelum dikirim
  • File: lib/meta-capi.ts → trigger setelah payment webhook sukses

5. Budget & Scaling Roadmap

📅 Phase 1: Testing (Minggu 1-2) — Rp 50.000/hari

Funnel%DailySetup
TOF Awareness70%Rp 35.0001 Campaign, 3-5 Ad Sets (beda audience), 2-3 creative per ad set
MOF Retarget20%Rp 10.0001 Campaign, 1-2 Ad Sets (IG engagers + web visitors)
BOF Conversion10%Rp 5.0001 Campaign, 1 Ad Set (cart + checkout abandonners)

Goals Phase 1

  • Test 6-10 creative variations (hook, angle, format berbeda)
  • Kumpulkan min 50 pixel events (ViewContent, AddToCart)
  • Identifikasi winning creative (CTR > 2%, CPC < Rp 3.000)
  • Build custom audiences dari pixel data
  • Target CPA: Rp 30.000 - 50.000 per purchase
Sabar! Di budget Rp 50rb/hari, Meta butuh waktu 3-5 hari untuk learning phase. Jangan matikan ad set sebelum 3 hari kecuali spend >2x target CPA tanpa konversi.

📅 Phase 2: Optimization (Minggu 3-4) — Rp 75.000-100.000/hari

Funnel%DailyAction
TOF60%Rp 45-60KKill losers, duplicate winners ke audience baru, test new creatives
MOF25%Rp 19-25KAdd video viewers audience, test carousel social proof
BOF15%Rp 11-15KDynamic retargeting, urgency creatives

Optimization Rules

  • Kill: Ad set dengan CPA > 2x target setelah 3 hari
  • Scale: Ad set dengan CPA ≤ target → naikkan budget 20% per 2-3 hari
  • Duplicate: Winning ad set → buat copy ke audience berbeda
  • Refresh: Creative fatigue setelah 7-10 hari (frequency > 2.5) → ganti hook/visual
  • Mulai buat LAL dari purchase data (kalau udah ≥ 50 purchases)

📅 Phase 3: Scaling (Minggu 5-8) — Rp 150.000-300.000/hari

Funnel%DailyAction
TOF55%Rp 83-165KLAL Purchasers 1% + 3%, expand geo (Kalimantan, Sulawesi), Advantage+
MOF25%Rp 38-75KFull retarget stack, sequential storytelling
BOF20%Rp 30-60KPromo codes, limited time offers, repeat buyer campaigns

Scaling Strategy

  • Horizontal scaling: Duplicate winning campaign ke audience segments baru
  • Vertical scaling: Naikkan budget 20-30% setiap 3 hari (jangan jump >50%)
  • Advantage+ Shopping Campaign: Setelah 100+ purchases, test ASC (biar algo full optimize)
  • CBO (Campaign Budget Optimization): Biarkan Meta alokasikan budget ke ad set terbaik
  • Target ROAS: Min 3x (spend 100rb → revenue 300rb)

📅 Phase 4: Maturity (Bulan 3+) — Rp 300.000-1.000.000/hari

  • Full Advantage+ Shopping Campaign sebagai main driver
  • Retarget purchasers untuk repeat order (upsell paket lebih tinggi)
  • Seasonal push: naikkan budget 2-3x saat season reset ML
  • Cross-platform: run parallel di TikTok Ads & Google Ads
  • Test influencer/KOL collab (micro-influencer ML 5-50K followers)
  • Target CPA: Rp 15.000-25.000 per purchase
  • Target ROAS: 4-6x

10. Leverage IG & FB Engagement

📱 Instagram @etnyx.id Strategy (Organic + Paid Synergy)

  • Post organic content 3x/minggu (sesuai 9-Post Grid plan) — ini jadi creative bank untuk ads
  • Boost best-performing organic posts → convert jadi ad dengan "Boost Post" atau create ad dari existing post
  • Stories daily: progress update, BTS push rank, promo countdown → build engagement audience
  • Reels first: 80% konten = Reels (reach paling besar di IG 2026)
  • CTA di setiap post: "Link di bio" atau "DM untuk order"
  • Engagement bait: Poll di Stories ("Main role apa?"), Quiz, Q&A

Organic → Paid Flywheel

  1. Post konten organic (Reels, Carousel)
  2. Monitor 24-48 jam — cek mana yang engagement paling tinggi
  3. Post dengan engagement tinggi → jadikan ad creative (proven content)
  4. Paid ads → drive traffic → sebagian follow IG → nambah organic reach
  5. More followers → more engagement → better engagement audiences → better retarget
  6. Repeat cycle ♻️
