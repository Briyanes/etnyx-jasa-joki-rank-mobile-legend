5. Step-by-Step: Customer Order

Apa yang customer lihat & lakukan dari awal sampai akhir

Step 1 — Pilih Paket
Buka etnyx.com/order
Pilih mode: Paket, Per-Star, atau Gendong
Lihat harga dari pricing catalog (CMS)
Opsi: Express (+20%), Premium (+30%)
Masukkan promo code (kalau ada)
Step 2 — Data Akun
Isi User ID + Server ID (Moonton)
Sistem verifikasi akun otomatis via Moonton API
Pilih login method (Moonton/FB/Google/TikTok/VK/Apple)
Masukkan login & password → dienkripsi AES-256
Request hero favorit (opsional)
Step 3 — Kontak & Add-ons
Isi nomor WhatsApp & email
Pilih opsi Express / Premium
Tulis catatan khusus untuk worker
Apply promo code / referral
Step 4 — Bayar
Auto Payment (iPaymu — QRIS, VA, e-wallet) → redirect, bayar, auto confirm
Manual Transfer → lihat rekening, transfer, upload bukti
Setelah order: Customer otomatis dapat WA + Email konfirmasi dengan link tracking — bisa pantau progress kapan saja tanpa tanya CS.

8. Panduan Per Role — Lead

Login: /admin → login dengan akun Lead

Melihat Order

  • Tampil order confirmed (belum assign) dan in_progress (tim sendiri)
  • Counter "Belum Ditugaskan" menunjukkan jumlah order pending
  • Pagination: 20 order per halaman

Assign Worker (Tugas Utama)

  1. Lihat order dengan status confirmed
  2. Klik "Tugaskan" pada order
  3. Pilih worker dari dropdown (hanya worker tim sendiri)
  4. Worker langsung dapat push notification + order muncul di dashboard-nya

Bulk Assign

  1. Centang beberapa order sekaligus
  2. Pilih worker → klik "Assign Semua"
  3. Diproses batch per 5 order

Submit Hasil

  1. Setelah worker selesai (status completed atau in_progress)
  2. Klik "Submit Hasil" pada order
  3. Upload screenshot before/after
  4. Customer otomatis dapat notifikasi + link review

Shared Notes & Worker Cards

  • Setiap order punya catatan bersama (Lead & Worker bisa tambah)
  • Lihat daftar worker: status aktif/idle, jumlah order aktif

9. Panduan Per Role — Worker

Login: /admin → login dengan akun Worker

Melihat Assignment

  • Tampil order yang di-assign ke kamu
  • Info: nickname customer, rank target, deadline

Mulai Kerja

  1. Klik order → lihat Credentials (akun game customer)
  2. Login ke game → mulai push rank
  3. PENTING: Jangan share credentials ke siapapun!

Update Progress

  1. Klik "Update Progress"
  2. Isi: Stars gained, MVP, Savage, Matches, Win, Duration, Screenshots
  3. Submit → customer langsung lihat update di tracking page
  4. Bisa edit submission dalam 30 menit

Selesaikan Order

  1. Rank target tercapai → klik "Selesaikan Order"
  2. Status berubah ke completed
  3. Lead akan upload hasil akhir

🚫 SOP Keamanan — WAJIB

DILARANG
✖ Akses email / data lain di akun customer
✖ Share credentials ke siapapun
✖ Pakai emulator yang terdeteksi
✖ Push di luar jam yang diizinkan
✖ Toxic / flame in game
WAJIB
✔ Login via secondary device
✔ VPN lokasi sesuai customer
✔ Play pattern natural (bukan bot)
✔ Screenshot setiap milestone

Yang TIDAK Bisa Dilakukan Worker:

10. Panduan Per Role — CS

Tool utama: WhatsApp Business + Dashboard (view only)

📲 Handle Chat Masuk
• Jawab DM dari sosmed / WA → SLA: < 5 menit
• Kualifikasi: tanya nama, rank sekarang, target rank
• Kasih info harga (refer ke pricing catalog)
• Guide customer ke etnyx.com/order
• Prinsip: arahkan ke self-service
🛡️ Handle Objection
• "Mahal" → Bandingin push sendiri vs 2-3 hari di ETNYX
• "Takut ban" → Enkripsi, VPN, play natural, 0 banned
• "Pikir-pikir" → Hold slot + harga 2 jam
• "Garansi?" → Star turun = gratis push ulang
🔄 Follow-up
• +2 jam: "Masih tertarik push rank?"
• +24 jam: "Promo masih berlaku hari ini"
• +3 hari: "Kalau butuh bantuan, ETNYX siap!"
• Max 3x follow-up, jangan spam
📞 Handle Komplain
• Cek status di tracking page
• Butuh action → eskalasi ke Leader / Admin
• Update progress → info "bisa cek di link tracking"
• Log setiap interaksi penting

12. Mode Order & Pricing

🎯 Paket (Rank → Rank)
Customer pilih paket (misal: Epic V → Mythic). Harga fixed per paket.
Contoh: Epic V → Mythic = Rp 315.089
⭐ Per-Bintang
Pilih rank + jumlah bintang. Harga per bintang × jumlah.
Contoh: Mythic Immortal × 3 = Rp 93.000
🤝 Gendong (Duo Boost)
Main bareng booster. Harga per bintang lebih tinggi. Tidak perlu share password.

13. Payment Flow

Transfer Manual (Selalu Aktif)

Customer pilih Manual→ /payment/manual→ Lihat rekening→ Transfer→ Upload bukti→ Admin approve

iPaymu Otomatis (Jika Aktif)

Customer pilih Auto→ Redirect iPaymu→ Bayar (QRIS/VA/e-wallet)→ Webhook auto-verify→ Confirmed ✅
Status iPaymu: Menunggu approval. Sementara hanya Transfer Manual.
Jika iPaymu gagal: Otomatis fallback ke Transfer Manual + customer dapat WA dengan info rekening.

24. FAQ Tim

Customer komplain?
Cek order di dashboard → lihat progress & logs → hubungi via WA. Serius? Eskalasi ke Admin.
iPaymu gagal saat bayar?
Otomatis fallback ke transfer manual. Customer dapat WA dengan info rekening.
Worker bisa handle >1 order?
Ya, worker bisa ditugaskan beberapa order sekaligus. Semua muncul di dashboard.
Tambah paket harga baru?
Admin → Settings → Pricing Catalog → Edit/tambah. Langsung aktif.
Customer cancel order?
Customer hubungi CS. Admin ubah status ke cancelled di dashboard.
Worker tidak responsif?
Lead bisa re-assign order ke worker lain melalui dashboard Lead.
