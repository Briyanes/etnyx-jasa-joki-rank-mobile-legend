🔤 Typography & Visual Rules

Font & Sizing
Inter — Clean, modern, readable

Hook text: Black 900, 36-48px, warna Accent (#2DD4BF)
Subtitle: SemiBold 600, 24-28px, White
Body: Regular 400, 18-20px, #E6F1EF
CTA button: Bold 700, 22px, di atas gradient bg
Visual Rules
🖤 Background SELALU dark (#0F1419)
✨ Subtle turquoise glow pada CTA
🎯 Max 3 elemen per screen
📐 Reels: 9:16 · Feed: 1:1 · Carousel: 4:5
🏷️ Logo ETNYX watermark kanan bawah
🎮 ML hero art sebagai visual utama
⚠️ Jangan: Background putih, warna neon selain palette, font script/decorative, terlalu banyak text per slide, screenshot tanpa overlay brand.

🖼️ Cover Komposisi — Template Layout

📱 Reels Cover (9:16 → crop 1:1 untuk feed)
Zona Aman Teks: Kiri atau tengah-atas, hindari 150px tepi kanan (area hero)
Hero/Character: Sudut kanan bawah, cut-out PNG no background, scale 80-100% tinggi frame
Glow Effect: Radial gradient warna aksen post di balik hero (opacity 15-20%)
Hook Text: Inter Black 900, 2-3 baris maks, warna putih + highlight aksen
Watermark: Logo ETNYX kanan bawah, ukuran 60-70px, opacity 90%
Sosmed icons: Baris bawah horizontal (IG, TT, FB, YT) — 12px, muted
🎨 Carousel Cover (4:5)
Slide 1 = Cover: Hero full kanan, teks kiri, badge format di atas
Slide 2-N: Tanpa hero, hanya teks + icon + warna aksen
Swipe hint: Arrow kanan bawah di slide 1, "Swipe →" di caption
Konsistensi: Header strip warna aksen 4px di atas setiap slide
Last slide: CTA full-gradient + logo besar + link
